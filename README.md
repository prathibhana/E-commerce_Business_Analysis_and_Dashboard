# 📊 E-commerce Data Analysis & Dashboard

This project focuses on analyzing e-commerce business performance using data analytics techniques and visualizing insights through an interactive dashboard.

## 🚀 Project Overview

The goal of this project is to evaluate key business metrics such as traffic sources, marketing campaigns, customer behavior, and overall sales performance. The analysis helps identify trends, inefficiencies, and opportunities to improve conversion rates and marketing ROI.

## 📅 Data Summary

- **Period:** Jan 1, 2023 – Mar 31, 2023  
- **Total Visitors:** 492,888  
- **Total Purchases:** 71,164  
- **Total Revenue:** $2.4M+  
- **Marketing Spend:** $1.14M  
- **Conversion Rate:** 5.64%  
- **Cart Abandonment Rate:** 29.53%  

## 🧹 Data Processing

- Validated data types for all columns  
- Handled missing values (Marketing Spend, Purchases, Revenue)  
- Replaced null values with mean values  
- Created a cleaned and transformed dataset for analysis  

## 📈 Key Analysis Areas

### 1. Sales & Revenue
- Consistent monthly revenue exceeding $700K  
- Revenue per visitor ~ $5  

### 2. Traffic Sources
- Highest traffic from Direct sources  
- Best conversion rate from Email Campaigns  

### 3. Marketing Campaigns
- Winter Promo had the highest spend  
- Holiday Offers delivered the highest conversion rate  

### 4. Customer Behavior
- New customers generated slightly more purchases than returning customers  
- Higher revenue per purchase observed from new customers  

### 5. Performance Metrics
- Conversion Rate: 5.64%  
- Repeat Purchase Rate: 17.65%  
- Cart Abandonment Rate: 29.53%

## 🧠 Key Insights

- Email campaigns are highly effective for conversions  
- Organic traffic has potential but needs optimization  
- Some campaigns consume high budget with lower ROI  
- Cart abandonment is a major area for improvement  

## 💡 Recommendations

- Optimize landing pages and checkout flow to increase conversions  
- Reallocate budget to high-performing campaigns  
- Implement retargeting and cart recovery strategies  
- Improve customer retention with personalized offers
